# Nodejs Microservices
This is a microservices app using the following technologies:

* Nodejs
  * Express
* Sequelize
* MySQL
* Docker, this for creating the databases containers

  * Db-Plans
  * Db-Subscriptions
  * Users
* Redis
  * Redis CLI
* JTW for Auth
* Passportjs
  * Passport JTW Strategy
* AMQP Protocol
  * CloudAMQP
* Express Gateway
